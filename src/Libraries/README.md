# Vendored Libraries

This tree holds third-party or experimental libraries that can be linked into
the IDE. Many folders are placeholders; document their status here so nested
code is easy to discover.

| Subdirectory | Status |
| --- | --- |
| [`PIP/`](PIP/README.md) | Placeholder for Python package installs (currently empty). |
| [`Python3/`](Python3/README.md) | Placeholder for embedding CPython modules. |
| [`iceeee/`](iceeee/README.md) | Scratch space for experiments (currently only `.DS_Store` artifacts). |
| [`stdbool/`](stdbool/README.md) | Compatibility shim intended for projects lacking `<stdbool.h>` (currently empty). |

Populate each subdirectory README when real code is added.
